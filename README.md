Online Laptop Shop is a dynamic e-commerce website developed using PHP & MySQL. The system allows users to browse laptops, add products to cart, place orders, and track order status. It also includes an admin panel to manage users, products, brands, orders, and reviews.

Features:
User:
User Registration & Login (Session Handling)
Browse & Filter Products
Add to Cart
Place Orders & Track Status
View Order History
Give Product Reviews
Edit Profile

Admin:
Admin Login
Manage Products (Brand-wise)
Manage Categories & Brands
Manage Users
View & Manage Orders
View Revenue Reports
Manage Reviews
